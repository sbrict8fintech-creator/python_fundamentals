# python_fundamentals

Mathematical and logical tools built from scratch in pure Python.

---

## Tools

| File | What it does |
|------|-------------|
| `combinations_maker.py` | Generates all possible combinations (Cartesian product) from given values — no itertools |
| `y_equation.py` | Finds the line equation given two points |
| `overheads.py` | Overhead cost allocator across departments |

## Philosophy

No libraries. Every algorithm written from first principles to understand the underlying logic.
